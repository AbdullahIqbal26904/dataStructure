Course Description: 

The purpose of this course is to provide the students with solid foundations in the basic concepts of programming: data structures and algorithms. The main objective of the course is to teach the students how to select and design data structures and algorithms that are appropriate for problems that they might encounter. This course is also about comparing algorithms and studying their correctness and computational complexity. This course offers the students a mixture of theoretical knowledge and practical experience using JAVA.

Course Objectives:

  • Learn basic data structures and performance measures using Big Oh analysis.
  • Learn to write recursive and iterative methods.
  • Learn the design and implementation of classic data structures such as linked lists, Array Lists, binary trees, searching and sorting methods, and their applications.
  • Learn the advanced data structures such as balanced search trees, hash tables, priority stack, queues, etc.
  • Learn various kinds of sorting algorithms that include comparison sorts, divide and conquer sorts and tree sort, etc.
  • Learn basic graph data structures and their algorithms
  • Learn to become a master in utilizing the standard data structure library of a major programming language (e.g. java.util in Java 5)
  • Learn to become a master in analyzing problems and writing program solutions to problems using appropriate data structures and algorithms.

 

Learning Outcome:

  1. CLO1: Understand classic data structures and their trade-off for efficient problem-solving.
  2. CLO2: Implementation of basic data structures and algorithms using a programming language.
  3. CLO3: Apply classic data structure and algorithms to solve real-world problems.

(Tentative) Detailed Weekly Plan

Week

Topics

1

Course Overview, discuss the course outline and grading policy.

Introduction to Arrays as linear data structure and understand its pros and cons using Linear and binary search algorithms.

2

Asymptotic Analysis (big-O notation) to measure the performance with respect to time and space complexity, understanding of Generic data types, and their importance in the data structure design.

3

Basic types of Linked lists (Singly, doubly, and circular) and their applications, performance measures using Big-oh analysis

4

Recursion (Linear recursion and binary recursion), its applications, and performance analysis

5

Stacks and its applications (infix to postfix conversion and postfix evaluation)

6

Types of Queues (simple queue, circular queue, priority queue), their applications, and performance analysis

7

Binary Trees construction and traversal methods performance measure and its applications.

8

Binary Trees deletion methods and its performance measures using Big-oh analysis.

9

Balanced trees include the AVL trees, their basic manipulation methods, and performance analysis

10

Searching methods include Sequential/linear search, binary search, indexed sequential search, tree search, and hashing technique. Hashing techniques, their application, and performance analysis with respect to different types of data.

11

Graph data structure and its applications

12

Sorting algorithms (bubble, insertion, selection, quick, heap and merge sort), their applications, and performance analysis

13

Balanced trees include a B-tree algorithm to handle large data

14
